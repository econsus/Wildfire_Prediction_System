Guide on using the csv_filter and combine_csv scripts

Only use on max 5 csv files at a time.

How to use:

1. only use with up to 1 day worth of files.
2. resulting in 1 file called combine_csv.csv
3. filter the combine_csv.csv
4. resulting in filtered_data.csv
5. save the filtered_data.csv to another folder.
6. switch the content of the csv_folder with the next 5 or 6 file batch.
